# Bhumi Raksha: Skies of Tomorrow

Game edukasi berbasis web bertema SDGs Goal 13 — Climate Action.

## Ringkasan

Bhumi Raksha adalah endless runner di mana pemain mengendalikan pesawat ramah lingkungan, menghindari obstacle, dan menjawab pertanyaan edukatif untuk menghancurkan obstacle khusus.

## Struktur Proyek

- `index.html` — Entry point
- `src/` — Kode sumber (scenes, utils)
- `assets/` — Gambar & audio
- `data/questions.json` — Bank soal (JSON)
- `docs/CONCEPT.md` — Dokumen konsep proyek

## Menjalankan Secara Lokal

Buka `index.html` di browser atau jalankan server statis, contohnya dengan Python:

```bash
# Python 3
python -m http.server 8000
# lalu buka http://localhost:8000
```
